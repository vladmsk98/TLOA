# TLOA EP Website

Минималистичный сайт для музыкального релиза TLOA EP.
Разработка: PYSCRIPT STUDIO, 2026.

## Особенности

Чистый, легковесный HTML/CSS/JS.
Прослушивание треков: воспроизведение/пауза по клику на обложку.
Просмотр текста трека: отображается во встроенном блоке под кнопкой (на светлом фоне с рамкой).
Возможность поделиться ссылкой на аудио через модальное окно с выбором платформы (ВК, Telegram, OK, Bluesky, копировать).
Навигация по трекам: кнопки в верхней части страницы для быстрого перехода.
Индикатор активного трека: обложка и кнопка "Текст трека" подсвечиваются при воспроизведении (сепиевый оттенок).
Авто-скролл к треку при открытии ссылки с якорем (например, #JAZZSTREETFLOW).
Поддержка светлой/тёмной темы.
Адаптивный дизайн.
Мета-теги для корректного отображения в соцсетях.

## Использование

Кликните на обложку трека, чтобы начать или поставить на паузу воспроизведение.
Используйте кнопку «Текст трека» под аудио, чтобы посмотреть лирику во встроенном блоке.
Используйте кнопки навигации в верхней части для быстрого перехода к нужному треку.
Используйте кнопку «Поделиться аудио» под аудио, чтобы поделиться ссылкой.
Используйте кнопку «Сменить тему» в правом верхнем углу.

## Совместимость

Протестировано в современных браузерах (Яндекс, Chrome, Firefox).
Функция копирования ссылки может не работать в старых браузерах.
